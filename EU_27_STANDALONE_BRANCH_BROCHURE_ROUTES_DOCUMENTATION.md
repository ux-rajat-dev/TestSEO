/>

### Dynamic Route Generation
```typescript
// Generate route for any EU country
const generateStandaloneBranchBrochureRoute = (countryAdjective: string) => {
  return `/${countryAdjective}-branch-brochure`;
};

// Usage examples
generateStandaloneBranchBrochureRoute('dutch');     // '/dutch-branch-brochure'
generateStandaloneBranchBrochureRoute('german');    // '/german-branch-brochure'
generateStandaloneBranchBrochureRoute('french');    // '/french-branch-brochure'
generateStandaloneBranchBrochureRoute('estonian');  // '/estonian-branch-brochure'
```

## Component Structure

### BranchBrochureLayout

- Dark theme styling (`bg-[#0F0B1F]`)

- Includes pricing, tax rates, and features
- Follows consistent design patterns
